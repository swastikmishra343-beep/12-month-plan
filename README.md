# 12 Month Plan

## Goal
Become internship-ready for software engineering roles with strong Java + DSA and basic web development skills.

## Current level
- Java: basic OOPs
- C: basic
- DSA: starting
- Web dev: starting
- AI/ML: not chosen yet

## Weekly time
- DSA: 75 minutes
- Web dev: 75 minutes
- Revision/GitHub: 15 minutes

## Why this plan
I want to build strong problem-solving skills, learn practical development, and create proof of work for internships.

## 12-month roadmap

### Month 1-2
- Revise Java basics and OOPs.
- Learn arrays and strings.
- Start Git and GitHub.
- Solve easy DSA problems.

### Month 3-4
- Learn recursion, linked lists, stacks, and queues.
- Start HTML and CSS.
- Build a simple static website.
- Push everything to GitHub.

### Month 5-6
- Learn trees, hash maps, and searching/sorting.
- Learn JavaScript basics.
- Make your website interactive.
- Continue solving DSA problems.

### Month 7-8
- Build one complete project.
- Add README, screenshots, and deployment link.
- Improve code quality and cleanup.

### Month 9-10
- Learn advanced DSA topics.
- Solve mixed practice problems.
- Start internship preparation.
- Update resume and LinkedIn.

### Month 11-12
- Apply for internships.
- Practice interview questions.
- Revise weak topics.
- Improve projects and proof of work.
